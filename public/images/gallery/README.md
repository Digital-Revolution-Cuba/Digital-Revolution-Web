# Gallery Images

Place gallery images in the appropriate subdirectory by category.

## Arte (`arte/`)

Expected: `arte-1.jpg` through `arte-8.jpg` (600×400 recommended)

## Música (`musica/`)

Expected: `music-1.jpg` through `music-8.jpg` (400×400 recommended, square for album covers)

## Fotografía (`fotografia/`)

Expected: `photo-1.jpg` through `photo-8.jpg` (600×400 recommended)

See `src/data/arteGallery.ts`, `src/data/musicaGallery.ts`, and `src/data/fotografiaGallery.ts` for the full list.

Until real images are added, cards will show broken images.
You can use placeholder images during development.
